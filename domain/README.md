# OpenDomain Knowledge

This directory is reserved for actual OpenDomain knowledge maintained by this
repository or by projects using this repository as a fixture.

Use these subdirectories:

- `contexts/`
- `concepts/`
- `rules/`
- `lifecycles/`
- `events/`
- `candidates/`

Do not write AI-inferred knowledge directly into accepted files. Put it under
`candidates/` first with evidence and review metadata.
